# دورة SQL 102 — Satr (Tuwaiq Academy)

هذا المستودع يحتوي على مشروع لمادة دورة (SQL 102) التي أنجزتها عبر منصة سطر التابعة لأكاديمية طويق، وهو مكمّل لمشروع (SQL 101).

## محتويات المستودع
- `MySQL 102 Course - Track.docx` — ملف المشروع بصيغة وورد.
- `MySQL 102 Course - Track.pdf` — ملف المشروع بصيغة PDF.

## كيفية العرض
1. لعرض المشروع بسهولة، افتح الملف `MySQL 102 Course - Track.pdf`.
2. يمكن أيضًا تحميل الملف وفتحه محليًا على أي قارئ PDF.


## محتوى المشروع
تم تنفيذ مجموعة من المهام التطبيقية باستخدام قواعد البيانات MySQL، من ضمنها:
* إنشاء جداول مشتقة من بيانات موجودة.
* استخدام Aggregate Functions مثل AVG, MAX, MIN.
* تطبيق String Functions وNumeric Functions.
* تنفيذ عمليات UPDATE, SELECT, وCREATE TABLE بشروط منطقية متنوعة.
* التعامل مع البيانات لتمييز الطلاب المتفوقين وغير المجتازين.
  
## ملاحظات
- قاعدة البيانات أنشئت باستخدام MySQL Workbench.
- تم توثيق جميع الخطوات بلقطات شاشة ضمن الملف المرفق.
